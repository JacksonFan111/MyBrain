Overview

Chris Set up these new servers and DB in 2022 to accommodate the needs

KAPSQLDAT-XX is the new primary server for all the various data-related functions at CIP. It carries on in SQLCIP\DataServices (Old Dumping ground) footsteps as a workhorse for all non-application DBs, primarily used to assist with integrations and assist the Data Warehouse.

Database

Primary Purpose

Integration

For all data in transit, i.e. staging data whilst it moves from one area to another or temporary data for a particular calculation or report/process.

NZXWT

Data dropped by NZXWT for use in the Data Warehouse (as we don't connect to them directly for BI)

ReferenceBI

A reference table to provide mappings for BI to marry up the two transactional systems as well as contains all the BI Security model information (used to build up the RLS in BI)

SSISDB

Used for SSIS Integrations

StagingBI

A staging environment for data destined for the Data Warehouse

Vault

A DB for storing data which is supposed to have history kept. Examples would be IRD reporting data, Funds Flow.
