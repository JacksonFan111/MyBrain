Jackson Fan : 14. Known Issues - Power BI colors changing when reports are published from Power BI Desktop to the web service

Created by Jackson Fan, last modified on 12-09-24

Microsoft has acknowledged the issue of Power BI colors changing when reports are published from Power BI Desktop to the web service. However, they have not provided a permanent fix as of the latest reports. Their primary guidance includes ensuring you are using the latest version of Power BI Desktop, as some updates have addressed specific visual-related bugs. Additionally, Microsoft suggests trying the following steps:

  1. Reverting to Default Colors: As mentioned earlier, one of the common fixes recommended by users and community members is to revert visuals to default colors in Power BI Desktop before publishing to the web service(

  2. Custom Themes: If you are using custom themes, Microsoft recommends reviewing the JSON structure of the theme to ensure it's correctly formatted and doesn't contain any problematic elements like wildcard characters. Simplifying the theme can sometimes resolve issues(

  3. Workarounds: One workaround suggested by the community, which Microsoft has referenced in support discussions, involves publishing a copy of the report after making changes to the original template. This sometimes helps bypass the color change issue

Step-by-Step Guide for New Users:

  1. Clear Cache on Power BI Desktop:

    • Open Power BI Desktop.

    • Navigate to the File menu in the top-left corner.

    • Go to Options and settings and select Options.

    • In the left-hand menu under Global, click on Data Load.

    • Find the option Clear cache and click on it to clear the cache data from Power BI Desktop.

  2. Delete the Report from Power BI Service:

    • Open your web browser and log in to the Power BI Service (https://app.powerbi.com).

    • In the Workspace where your report is located, find the report you want to delete.

    • Select the report, click the More options (three dots) next to the report name, and choose Delete.

    • Confirm the deletion when prompted.

  3. Re-upload the Report to Power BI Service:

    • Go back to Power BI Desktop.

    • After clearing the cache, re-open your report or ensure the latest version is open.

    • Click on Publish in the top-right corner of the Power BI Desktop window.

    • Select the appropriate Workspace in Power BI Service and click Select to upload the report again.

  4. Verify the Fix:

    • Once the report is re-uploaded, go to the Power BI Service in your web browser.

    • Check the colors in the visuals to ensure they now match the colors seen in Power BI Desktop.

This should resolve the color differences between the Power BI Desktop and Power BI Service.
