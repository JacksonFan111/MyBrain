Certainly! Here’s a simplified step-by-step solution to create a dynamic free-text visual in Power BI using SmartSheet and two table visuals—one for headers and one for body text. This method gives you flexibility and provides a clean layout for dynamic content.

Step 1: Set Up the SmartSheet for Headers and Body Text

  1. Create a SmartSheet Table:

    • You need two main fields in your SmartSheet:

      1. Header: A field for the section headers (e.g., "Cybersecurity Update").

      2. BodyText: A field for the free-text body (e.g., detailed content related to the update).

    • You can have multiple rows in the SmartSheet representing different sections or updates.

  2. Share the SmartSheet:

    • Make sure the SmartSheet is shared with the necessary team members who can add or update content.

    • Set them as Editors so they can input both header and body text.

Step 2: Import SmartSheet Data into Power BI

  1. Connect to SmartSheet:

    • In Power BI Desktop, go to Home > Get Data > Web and use the SmartSheet API link to pull in your data, or export your SmartSheet as a CSV or Excel file.

    • Once the connection is set, import the SmartSheet table with the Header and BodyText fields.

  2. Load the Data:

    • After connecting, load the SmartSheet data containing the header and body text fields into Power BI.

Step 3: Create Two Table Visuals

1. Header Visual

  • Add a Table Visual for the Header:

    • In Power BI Desktop, from the Visualizations pane, add a Table visual.

    • In the Fields pane, drag the Header column from the SmartSheet data into the Values field of the Table visual.

    • Formatting:

      • Go to the Format pane and under Grid, remove the gridlines and borders.

      • Set the background color, font size, and alignment to make it visually appealing and easily distinguishable.

      • You may also enable word wrap if your headers are long.

2. Body Text Visual

  • Add a Second Table Visual for the Body:

    • In the Visualizations pane, add another Table visual for the body text.

    • Drag the BodyText column into the Values section.

    • Formatting:

      • Similarly, format the table with no borders, no gridlines, and enable word wrap to handle long text.

      • Adjust the font size and style to ensure the body text is easy to read and matches the rest of your dashboard’s design.

Step 4: Apply Rounded Corners and Clean Design

  1. Insert a Rounded Rectangle Shape:

    • Go to Insert > Shapes > Rectangle and set the corner radius to create rounded corners.

    • Set the Fill Color to complement your dashboard’s color theme and adjust transparency if necessary.

    • Place this shape behind both table visuals to give them a polished, cohesive look.

    • Resize the shape to cover both the header and body text sections.

  2. Send Shape to the Back:

    • Right-click the shape and select Send to Back to ensure the text visuals are on top of the rounded corner shape.

Step 5: Dynamic Updates via SmartSheet

  1. Ensure Refresh is Configured:

    • Set up a scheduled data refresh in Power BI to ensure any updates in the SmartSheet are reflected in the Power BI visuals.

    • Go to the dataset settings and configure the refresh interval (e.g., every hour or as needed).

  2. Allow Manual Refresh:

    • Optionally, you can add a Refresh Button (using Power Automate) to let users manually refresh the data in real-time.

Step 6: Final Adjustments and Testing

  1. Adjust Visual Sizes and Layout:

    • Ensure the header and body text visuals are sized appropriately so they fit within the rounded corner shape and the overall dashboard layout.

  2. Test Dynamic Updates:

    • Make a test update to the SmartSheet (both the header and body text) to ensure that the changes are reflected in Power BI after the refresh.

End Result

  • You now have a dynamic free-text visual in Power BI where:

    • The Header (e.g., "Cybersecurity Update") is displayed in one table visual.

    • The BodyText (detailed description) is displayed in a second table visual below the header.

    • The layout includes a rounded corner background for a clean and modern look.

    • Real-time updates from the SmartSheet are reflected automatically in Power BI.

This setup allows for a clean and dynamic display of text that can be easily updated by end users in SmartSheet, with changes reflected in Power BI.
